# "Auto Service NISHTYAK" - Car Service Automation Application

**"Auto Service NISHTYAK"** is a software solution for automating car service operations. It allows for the management of customers, vehicles, orders, and spare parts, simplifying the organization of work and improving efficiency.

## Main Features

- **Customer and Vehicle Management**: Keep track of clients and their vehicles.
- **Order Creation and Management**: Create and manage service orders.
- **Spare Parts and Inventory Management**: Track spare parts and warehouse stock levels.
- **Employee Assignment to Orders**: Assign employees to specific orders.
- **Report and Analytics Generation**: Generate reports and analyze data for better decision-making.

## Project Structure

- **Backend**: Java, Spring Boot - handles business logic and communicates with the database.
- **Database**: PostgreSQL - stores customer, order, and spare part data.
- **Frontend**: Simple interface or command-line application for user interaction.

## Build:

`$ mvn clean package`

## Run:

`$ mvn spring-boot:run`
